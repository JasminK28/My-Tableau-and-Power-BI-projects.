# My-Tableau-and-Power-BI-projects.
📊 Tableau & Power BI Data Analysis Project
Data Technician Bootcamp Portfolio
This project showcases the practical work completed during a Data Technician bootcamp, focusing on data analysis, visualisation, and dashboard development using Tableau and Power BI. The goal was to transform raw datasets into meaningful insights that support data-driven decision-making.

📁 Project Overview
Throughout this project, I worked with real-world datasets to explore trends, build interactive dashboards, and communicate insights clearly.
🌍 Global Health Analysis (Tableau)
Analysed an international health dataset to identify patterns in life expectancy, population, and health indicators.
Visualisations created:

Bar charts to compare life expectancy across continents
Line charts to track trends over time
Pie charts to show population distribution by gender
Scatter plots to explore relationships between health metrics
Interactive dashboard combining all insights

Key outcomes:

Identified disparities in life expectancy between regions
Analysed long-term health trends across countries
Explored relationships between variables such as BMI and life expectancy


🎵 Spotify Data Analysis (Tableau)
Explored a music dataset to analyse genre popularity and track characteristics.
Insights generated:

Comparison of genre popularity
Relationships between audio features (e.g., energy vs danceability)
Variation in musical mood (valence) across genres

Deliverable:

Fully interactive Tableau dashboard highlighting trends and patterns
